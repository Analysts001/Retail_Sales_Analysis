# Retail Sales Analysis

## Overview
This project analyzes the **Superstore sales dataset** using Python (pandas, matplotlib, seaborn) to find insights on sales, profit, and top states.

## Dataset
- File: Superstore.csv
- Rows: 9994
- Columns: 21
- Source: Kaggle

## Project Structure
Retail_Sales_Analysis/
├─ retail_sales_analysis.py
├─ Superstore.csv
├─ graph1_sales.png
├─ graph2_profit.png
├─ graph3_top_states.png
├─ output_terminal.txt
└─ README.md

## How to Run
1. Install libraries: `pip install pandas matplotlib seaborn`  
2. Run: `python retail_sales_analysis.py`  
3. Check graphs and output.
